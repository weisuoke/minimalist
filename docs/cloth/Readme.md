- 阿美咔叽/美式复古

  - 选择的衣服易于打理、维护
  - 品牌选择：Freewheelers、Toys Mccoy、The Real Mccoy
- 户外
  - 轻量化的功能性服饰。
  - 适用场景： 恶劣天气、外出游玩
  - 选择品牌：始祖鸟

    - 裤子：Denim为主、其次卡其色
